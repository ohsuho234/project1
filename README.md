# HY Portal 리디자인 프로젝트  
한양대학교 포털 시스템 UI/UX 개선 웹사이트

## 📌 1. 프로젝트 소개
이 프로젝트는 한양대학교 재학생들이 일상적으로 사용하는 포털 시스템을 보다 직관적이고 효율적인 형태로 재구성하기 위해 진행된 리디자인 웹 프로젝트입니다.  
기존 포털에서 사용자들이 겪는 불편 요소들을 개선하고, 핵심 기능(시간표, 공지, 식단, 비교과, 취업 정보 등)을 한 화면에서 빠르게 접근할 수 있도록 정보 구조(Information Architecture)와 사용자 흐름(User Flow)을 새롭게 설계하였습니다.

실제 HTML/CSS 기반으로 구현되었으며, GitHub Pages를 통해 웹페이지 형태로 배포하여 누구나 직접 이용해볼 수 있습니다.

👉 **데모 사이트:** https://ohsuho234.github.io/project1/

---

## 📌 2. 프로젝트의 필요성
현행 한양대학교 포털은 기능은 매우 방대하지만, 다음과 같은 문제점으로 인해 사용자들이 원하는 정보를 빠르게 찾기 어렵습니다.

- 여러 기능이 서로 다른 페이지에 분산되어 있어 **접근성이 낮음**
- UI 구성 요소의 **일관성이 부족**하여 메뉴 구조가 복잡하게 느껴짐
- 모바일 및 태블릿 환경에서 **사용성이 떨어지는 문제**가 지속적으로 제기됨
- 반복적으로 사용하는 기능(시간표, 식단, 주요 공지 등)을 **한 화면에서 확인할 수 없음**
- 초보 사용자(신입생) 관점에서 익숙해지기까지 시간이 오래 걸림

이러한 문제들은 재학생 전반의 학습·행정 효율성에도 직접적인 영향을 주기 때문에, 사용자 중심의 UI/UX 개선이 필요한 상황이었습니다.

---

## 📌 3. 프로젝트의 중요성
이 리디자인 프로젝트는 단순한 시각적 변경이 아니라, **사용자가 실제로 어떤 기능을 가장 많이 사용하며, 무엇이 접근성을 떨어뜨리는지 분석한 결과물**입니다.

이를 통해:

- 핵심 기능을 중심으로 재배치하여 **정보 접근 속도 향상**
- 재학생들이 매일 사용하는 포털의 **실질적인 편의성 개선**
- UI 구성 요소를 공통 규칙에 따라 정리하여 **일관된 사용자 경험 제공**
- 신규 사용자도 쉽게 적응할 수 있는 **직관적 구조** 구현
- 실제 서비스 개선 제안서로 활용 가능하며, 포트폴리오 가치가 높음

---

## 📌 4. 주요 기능 & 화면 구성
- 메인 홈 화면 리디자인  
- 주요 메뉴(학사, 장학, 취업, 비교과 등) 접근성 향상  
- 캘린더 및 학사 일정 표시  
- 빠른 메뉴 바로가기 카드 UI  
- 학식/식단 정보 컴포넌트  
- 전체 화면 UI 구성 및 컬러 시스템 통일  
- 반응형 웹 기반 레이아웃 구성  

---

## 📌 5. 기대 효과
- 학생들의 포털 탐색 시간을 줄이고 편의성 증가  
- 기능 분산 문제 해결로 빠르게 목적 기능 접근 가능  
- 사용자 중심 디자인으로 **UX 품질 향상**  
- 재학생·신입생 모두에게 직관적인 사용자 경험 제공  
- 실제 시스템 개선에 참고 가능한 리디자인 시안으로 활용 가능
